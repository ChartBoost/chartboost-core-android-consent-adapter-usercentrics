# Chartboost Core Consent Usercentrics Adapter

The Chartboost Core Usercentrics adapter mediates Usercentrics SDK via the Chartboost Core SDK.

## Minimum Requirements

| Plugin              | Version |
|---------------------|---------|
| Chartboost Core SDK | 0.4.0+  |
| Android API         | 21+     |

## Integration

In your `build.gradle`, add the following entry:
```
    implementation "com.chartboost:chartboost-core-adapter-usercentrics:0.2.8.1.5"
```
